# 🛒 Online Shop Web Application

Бұл жоба — backend және дерекқоры бар қарапайым онлайн дүкен веб-қосымшасы.  
Жоба оқу мақсатында жасалған және CRUD, аутентификация және дерекқор байланысын қамтиды.

---

## 🚀 Project Features

- 🔐 User Registration & Login
- 🛍️ Product CRUD (Create, Read, Update, Delete)
- 🔎 Search & Filter Products
- 🗂️ Categories system
- 🗄️ Relational Database (PostgreSQL / MySQL)
- ⚙️ Backend API (Node.js + Express)
- 📊 ER Diagram (Database structure)

---

## 🧑‍💻 Technologies Used

- Node.js
- Express.js
- PostgreSQL
- JavaScript
- HTML / CSS (optional frontend)
- Git & GitHub
